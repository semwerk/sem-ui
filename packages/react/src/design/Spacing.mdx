# Spacing System

Consistent spacing scale for margins, padding, and gaps.

## Scale

The spacing system uses a 4px base unit with a geometric progression.

<div style={{ display: 'grid', gap: '1rem', marginTop: '2rem' }}>

## Usage Guidelines

| Scale | Usage |
|-------|-------|
| 1-3 | Tight spacing (badges, compact lists) |
| 4-6 | Component padding, small gaps |
| 8-12 | Section spacing, card padding |
| 16-24 | Page sections, major layout gaps |

## Examples

```tsx
// Component padding
<div style={{ padding: 'var(--space-4)' }}>...</div>

// Stack layout
<div style={{ display: 'flex', flexDirection: 'column', gap: 'var(--space-6)' }}>
  <div>Item 1</div>
  <div>Item 2</div>
</div>

// Section spacing
<section style={{ marginBottom: 'var(--space-12)' }}>...</section>
```
