# Werk Design System

This component library uses the **@werk-design** token system for consistent styling across Werkcontext and WerkPress.

## Design Packages

- **@werk-design/tokens** - Core design token definitions (colors, spacing, typography)
- **@werk-design/css** - CSS implementations with light/dark themes
- **@werk-design/tailwind-preset** - Tailwind CSS integration

## Themes

The design system supports multiple themes:

### Base Themes
- `base.light.css` - Default light theme
- `base.dark.css` - Default dark theme

### Product Themes
- `werkcontext.light/dark.css` - Werkcontext-specific styling
- `werkpress.light/dark.css` - WerkPress-specific styling

## Using Design Tokens

All components in this library use design tokens automatically through the imported CSS.

Toggle between light/dark themes using the toolbar background selector.
